Projeto ColdFusion

Sobre o Projeto

Este projeto demonstra como usar componentes ColdFusion (.cfc) para gerenciar usuários em um banco de dados.

Como Configurar

  1. Configure um banco de dados MySQL chamado test_db com a tabela users.
  2. Faça o deploy do código em um servidor ColdFusion.
  3. Acesse o arquivo index.cfm para interagir com o projeto.
